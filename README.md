## ผู้สมัครหลัก (อันดับความนิยมปัจจุบัน)

### 1. ประชาชาติ แสนแก้ว (ตุง) - เพื่อไทย
- **คะแนน**: 32% (+3 จากสัปดาห์ที่แล้ว)
- **จุดเด่น**: มรดกครอบครัวทางการเมือง, เครือข่ายเสื้อแดง, นโยบายช่วยเหลือเกษตรกร
- **ประเด็นหลัก**: ราคาอ้อย 1,200-1,300 บาท/ตัน, ชดเชยรายได้เกษตรกร

### 2. ธนอนันต์ เมนะสวัสดิ์ (ทนายโตโต้) - กล้าธรรม  
- **คะแนน**: 28% (+5 จากสัปดาห์ที่แล้ว)
- **จุดเด่น**: ความเชี่ยวชาญเทคโนโลยี, นโยบายเกษตรอัจฉริยะ, ภาพลักษณ์นักบริหารรุ่นใหม่
- **เป้าหมาย**: ผู้สมัครหลักที่ระบบติดตาม

### 3. อดิศักดิ์ แก้วมุงคุณทรัพย์ (ดูดี) - เพื่อไทย
- **คะแนน**: 25% (-2 จากสัปดาห์ที่แล้ว)  
- **จุดเด่น**: ส.ส.ปัจจุบัน, ประสบการณ์ท้องถิ่น, เครือข่ายครอบครัว
- **ความท้าทาย**: การเปลี่ยนพรรคจากไทยสร้างไทย

### 4. สรวิชญ์ นาแพงสอน - รวมไทยสร้างชาติ
- **คะแนน**: 18% (+1 จากสัปดาห์ที่แล้ว)
- **จุดเด่น**: แบรนด์พรรคก้าวหน้า, ฐานเสียงเยาวชน

## ข้อมูลประวัติศาสตร์ - การเลือกตั้ง 2566

- **รองชนะเลิศ**: จุฑาพัชร เมนะสวัสดิ์ (เพื่อไทย) - 22,885 คะแนน (28.1%)
- **อันดับ 3**: อนันต์ อมรินทร์ (ก้าวไกล) - 22,150 คะแนน (27.2%)
- **คะแนนรวม 3 อันดับแรก**: 89.4% ของคะแนนทั้งหมด

## โครงสร้างข้อมูล
- **โมเดลหลัก**: Candidate interface ใน `/src/data/candidates.ts`
- **ข้อมูลเสริม**: Historical data, district coverage, campaign intelligence ใน `/src/data/enhancedCandidates.ts`
- **โมดูล UI**: React components ใน `/src/components/modules/`

## ฟีเจอร์หลัก
1. **ภาพรวม (Overview)**: แดชบอร์ดหลักแสดงสถิติสำคัญ
2. **ข่าวกรองการเมือง (Intelligence)**: SWOT analysis, historical context, campaign tracking
3. **ฐานข้อมูลผู้สมัคร (Candidates)**: จัดการและค้นหาข้อมูลผู้สมัคร
4. **ติดตามสื่อสังคม (Social)**: การติดตามกิจกรรมโซเชียลมีเดีย
5. **วิเคราะห์ฐานเสียง (Demographics)**: ข้อมูลประชากรและการกระจายตัว
6. **ติดตามกิจกรรม (Activities)**: การติดตามกิจกรรมหาเสียง
7. **เปรียบเทียบและพยากรณ์ (Analysis)**: การวิเคราะห์เชิงเปรียบเทียบ

## เทคโนโลยี
- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Database**: Supabase (configured แต่ยังไม่ใช้งานจริง)
- **Build**: Vite with TypeScript
- **Deployment**: Cloudflare Pages

## การติดตั้งและรัน
```bash
npm install
npm run dev      # Development server
npm run build    # Build for production
npm run preview  # Preview production build
```

## การ deploy ไป Cloudflare Pages
```bash
npm run build
npx wrangler pages deploy dist --project-name candidate
```

## ประเด็นสำคัญในการเลือกตั้งครั้งนี้
1. **วิกฤตราคาอ้อย**: ราคาตกต่ำส่งผลกระทบต่อเกษตรกร
2. **การเปลี่ยนพรรคของผู้สมัครปัจจุบัน**: อดิศักดิ์เปลี่ยนจากไทยสร้างไทยเป็นเพื่อไทย

4. **เครือข่ายครอบครัว**: บทบาทของธีรชัย แสนแก้วในการสนับสนุนลูกชาย
